## Описание программы
** «Book Tracker» - простое приложение для учёта прочитанных книг. Позволяет добавлять книги с указанием названия, автора, жанра и количества страниц. Реализована фильтрация по жанру и количеству страниц. Данные сохраняются в файл `books.json`.**
## Требования для запуска
- Python 3.10 или выше
- Библиотеки: `tkinter` — для графического интерфейса

## Примеры использования
1. Команда запуска `python main.py`
 2. Добавьте книгу через форму и нажмите «Добавить книгу».
3. Используйте фильтры для поиска книг по жанру или количеству страниц.
4. Сохраняйте и загружайте данные через соответствующие кнопки.
